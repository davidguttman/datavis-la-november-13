Data Visualization with JavaScript
==================================

To run:

1) Install [Node.js](http://nodejs.org/) (> v0.10 should be fine)

2) From the code directory run
    
    npm install
    
3) Install Beefy
    
    npm install -g beefy

4) Run

    beefy index.js
    
5) Open up your browser to [http://localhost:9966](http://localhost:9966/)
